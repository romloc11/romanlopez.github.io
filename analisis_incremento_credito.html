Análisis Automatizado para Incremento de Crédito

Tecnologías: Python · Pandas · NumPy · Excel · OpenPyXL

Contexto del problema

La empresa contaba con una matriz extensa de clientes con información histórica de ventas, pagos, límites de crédito, comportamiento de cartera y clasificaciones internas. El análisis para decidir incrementos o decrementos de crédito se realizaba de forma manual, utilizando reglas dispersas y criterios subjetivos.

Esto generaba procesos lentos, inconsistencias en la toma de decisiones y un alto riesgo operativo al no contar con un criterio estandarizado y auditable.

Solución implementada

Se desarrolló un motor de análisis en Python que evalúa automáticamente el comportamiento de todos los clientes mediante un sistema de scoring crediticio, integrando información histórica de ventas, pagos, antigüedad, uso de línea, variabilidad y cumplimiento.

El sistema clasifica a cada cliente en una de las siguientes decisiones:

  • Incremento
  • Posible incremento (revisión manual)
  • Sin cambio
  • Posible decremento (revisión manual)
  • Decremento

Modelo de scoring

El score final se construye a partir de múltiples componentes ponderados, entre ellos:

  • Uso histórico del límite de crédito
  • Clasificación interna de comportamiento (ADN)
  • Días promedio de pago
  • Antigüedad del cliente
  • Variabilidad de ventas
  • Capacidad máxima de pago registrada
  • Estado de cartera vencida

Cada variable aporta al score final mediante pesos configurables, permitiendo flexibilidad y adaptación a políticas de riesgo.

Cálculo del monto sugerido

Además de la decisión crediticia, el sistema calcula un monto sugerido de crédito considerando:

  • Promedio y máximo de ventas de los últimos 12 meses
  • Límite actual del cliente
  • Cartera vigente
  • Score obtenido y tipo de decisión
  • Restricciones de crecimiento y protección de riesgo

El resultado es un monto redondeado y protegido, alineado a la capacidad operativa real del cliente.

Dashboard y resultados

El script genera automáticamente un archivo Excel con:

  • Base detallada de evaluación crediticia
  • Resumen ejecutivo con KPIs clave
  • Gráficas automáticas de decisiones de crédito
  • Comparativo entre límite actual vs límite sugerido

Este dashboard permite a las áreas comerciales y financieras visualizar rápidamente el impacto de las decisiones propuestas.

Impacto

  • Estandarización del proceso de análisis crediticio
  • Reducción significativa de tiempos de evaluación
  • Menor dependencia de análisis manual
  • Mejor control del riesgo crediticio
  • Decisiones más objetivas y basadas en datos

Nota sobre los datos

Los datos utilizados en este proyecto han sido anonimizados y/o simulados con fines demostrativos. La lógica, reglas de negocio y metodología reflejan un caso real de análisis crediticio empresarial.
