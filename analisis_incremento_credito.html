Análisis Automatizado para Incremento de Crédito

Tecnologías: Python · Pandas · NumPy · Excel · OpenPyXL

Ver código en GitHub

Contexto

La empresa contaba con una matriz extensa de clientes con información histórica de ventas, pagos, límites de crédito y comportamiento de cartera. El análisis se realizaba manualmente con criterios dispersos.

Problema

El proceso era lento, poco estandarizado y con alto riesgo operativo, dificultando decisiones consistentes y auditables.

Solución

Se desarrolló un motor de análisis en Python que evalúa automáticamente a cada cliente mediante un sistema de scoring crediticio.

  • Incremento
  • Posible incremento (revisión manual)
  • Sin cambio
  • Posible decremento (revisión manual)
  • Decremento

Modelo de scoring

  • Uso histórico del límite de crédito
  • Clasificación interna (ADN)
  • Días promedio de pago
  • Antigüedad del cliente
  • Variabilidad de ventas
  • Capacidad máxima de pago
  • Cartera vencida

Impacto

  • Estandarización del análisis crediticio
  • Reducción significativa de tiempos
  • Menor dependencia de análisis manual
  • Decisiones objetivas basadas en datos
